## Project Overview
This is an interactive Streamlit dashboard for visualizing UK housing data from 1995. The dashboard allows users to upload CSV files and explore trends in house prices, sales, and crime data across different areas.

## Features
- **Interactive Charts:** Built with Plotly for price trends and comparisons
- **Data Filtering:** Filter by area, date range, and price
- **CSV Upload:** Upload your own housing dataset
- **Responsive Table:** View raw data with sorting and search

## Tech Stack
- **Python 3.9**
- **Streamlit:** For the web app framework
- **Pandas:** For data manipulation
- **Plotly:** For interactive visualizations
